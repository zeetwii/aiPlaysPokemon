# AI Plays Pokemon

A while ago several of the major frontier model makers did a thing where they showed off using their cloud based models to complete different pokemon games.  It was an interesting test in concept, but I feel like their implementation missed the point, since complex tasks like playing a video game are as much a test of the model as they are the tooling that the model uses.  Thus this project: a set of open source tools to try and show even the smallest local models can play pokemon with the right tooling and context controls.  

## The Challenge

My goal with making all of these tools was to see what's the smallest local model that can beat pokemon.  The objective is to have both the models and tools all running locally on a laptop or desktop.  This means no internet based tooling to just do web searches or using externally hosted models.
